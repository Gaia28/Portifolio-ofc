Aplicativo de Anotações - Tema Hello Kitty

Este projeto é um aplicativo mobile de anotações desenvolvido para Android, com um visual totalmente personalizado no tema Hello Kitty. O objetivo é proporcionar uma experiência divertida e funcional para quem deseja organizar suas ideias, tarefas e lembretes de forma prática e estilosa.

O app permite criar, visualizar, editar e excluir anotações rapidamente. A interface é amigável, com cores suaves e elementos visuais inspirados na personagem Hello Kitty, tornando o uso mais agradável, principalmente para fãs do universo kawaii.

Tecnologias utilizadas: Java para o desenvolvimento Android, SQLite para armazenamento local das notas e XML para a construção das telas.

Tela de Criação de Anotações

A tela de criação de anotações permite que o usuário insira o título e o conteúdo da nota. Com um design intuitivo, é fácil adicionar novas ideias ou lembretes. O tema Hello Kitty está presente em todos os elementos, tornando a experiência única e divertida.

O usuário pode salvar suas anotações com um simples toque, garantindo que nenhuma ideia seja perdida.

Tela de Visualização de Anotações

Nesta tela, o usuário pode visualizar e excluir todas as suas anotações salvas. Cada nota é apresentada com seu título e a data em que foi criada, facilitando a navegação.

O design é otimizado para que o usuário encontre rapidamente o que procura, mantendo a estética Hello Kitty em cada detalhe.
