Gerenciador de Senhas

Este projeto é uma aplicação web desenvolvida para ajudar usuários a armazenar, organizar e gerenciar suas senhas de forma prática e segura.

Funcionalidades principais:
• Cadastro e login de usuários com validação.
• Adicionar senhas associadas a diferentes sites e serviços.
• Visualizar e organizar credenciais cadastradas.
• Visualização segura das senhas.

Tecnologias utilizadas: PHP (backend e sessões), HTML (estrutura), CSS (estilização), MySQL (armazenamento de dados).

Adicionando Senhas

O usuário pode cadastrar novas senhas associando-as a diferentes sites ou serviços. O formulário é simples e seguro, garantindo que as informações sejam armazenadas corretamente no banco de dados.

O sistema permite organizar as credenciais de acordo com a necessidade do usuário.

Senhas Salvas

Todas as senhas cadastradas ficam disponíveis para visualização em uma interface organizada. O usuário pode consultar, editar ou excluir suas credenciais conforme necessário.

A visualização das senhas é protegida, garantindo maior segurança ao usuário.

Visualizando Senha

O sistema permite que o usuário visualize a senha cadastrada de forma segura, exibindo-a apenas quando solicitado. Isso garante privacidade e proteção das informações sensíveis.

O design é simples e funcional, focado na usabilidade e segurança.

Tela de Login

A tela de login permite que o usuário acesse sua conta de forma segura, utilizando validação de credenciais. O sistema garante a proteção dos dados e impede acessos não autorizados.

O layout é simples e intuitivo, facilitando o acesso ao sistema.

Tela de Cadastro

A tela de cadastro possibilita que novos usuários criem uma conta informando seus dados básicos. O processo é rápido e seguro, com validação dos campos para garantir a integridade das informações.

Após o cadastro, o usuário pode acessar todas as funcionalidades do gerenciador de senhas.
